# Glowy
## What is it ?
Glowy is a 20*15 neopixel matrix running the WLED firmware.
## Why did I make it ?
I have always been fond of mood and party lighting, and I think that the idea of a matrix combines both of them.
## Features
- 90W Total output
- WiFi connectivity
- More than 100 lighting effect
- Sound reactive effects
